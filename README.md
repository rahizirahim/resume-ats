# 📄✨ ATS Resume Optimizer

An AI-powered web app that transforms your resume into an ATS-friendly format instantly.

## 🌐 Live Demo


## 🤖 How It Works
1. Upload your resume in PDF format
2. AI analyzes and optimizes your resume
3. Download your ATS-friendly resume in Word format

## ✅ Features
- AI-powered resume optimization using Llama 3.3
- Clean professional formatting
- Photo placeholder included
- Proper ATS-friendly structure
- Sections: Summary, Education, Experience, Skills

## 🛠️ Tech Stack
- **Python** + **Flask** (backend)
- **Groq API** + **Llama 3.3** (AI)
- **pdfplumber** (PDF extraction)
- **python-docx** (Word generation)
- **HTML/CSS/JS** (frontend)

## ⚙️ Setup & Installation

### 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/resume-ats.git
cd resume-ats
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Create .env file

GROQ_API_KEY=your_groq_api_key_here

### 4. Run the app
```bash
python app.py
```

### 5. Open in browser

http://localhost:5000

## 📌 Notes
- Only PDF files supported
- Works best with text-based resumes
- Scanned PDFs not supported
